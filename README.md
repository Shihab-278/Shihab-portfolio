Shihab Uddin — Portfolio

A modern, professional portfolio for Shihab Uddin (BSc in CSE, UITS — CGPA 3.91, 2020–2024). It showcases skills, projects, experience, and contact details with a contemporary UI, smooth interactions, and mobile-first design.

Live Demo
- URL: https://shihab-278.github.io/Shihab-portfolio/

Features
- Modern UI: Glassmorphism, gradient accents, and professional typography.
- Responsive Layout: Seamless experience on mobile, tablet, and desktop.
- Smooth Interactions: Scroll progress bar, intersection-based animations, micro-interactions.
- Mobile Navigation: Sticky glass navbar with mobile menu.
- Projects Showcase: Detailed cards with tech stack badges and links.
- Contact Form: Formspree integration for direct email submissions.
- SEO Ready: Meta tags, Open Graph, and accessible structure.

```bash
Setup Instructions

Clone the Repository:git clone https://github.com/shihab-278/shihab-portfolio.git


Open Locally:
Install the Live Server extension in VS Code.
Open index.html and click "Go Live" to view the website.


Deploy to GitHub Pages:
Push the code to a GitHub repository.
Go to Settings > Pages in the repository.
Select the main branch and / (root) folder.
The website will be live at (https://shihab-278.github.io/Shihab-portfolio/)

# Preview locally
python3 -m http.server 8000
# Open http://localhost:8000 in your browser
```
Alternatively, use VS Code Live Server to open `index.html`.

Deployment (GitHub Pages)
1. Push code to the `main` branch.
2. In GitHub repo: Settings → Pages → Source: `main` and folder `/` (root).
3. Your site will be live at: https://shihab-278.github.io/Shihab-portfolio/

Customization
- Profile Image: Update paths in `index.html` and `pages/projects.html` (e.g., `assets/my.jpg`).
- Projects: Edit cards in `pages/projects.html` (titles, tech tags, GitHub links, demos).
- Social Links: Update GitHub/LinkedIn/Twitter in the footer in both pages.
- Favicon: Place your icon at `assets/images/favicon.ico`.
- Contact Email: Formspree action is set in `index.html` (`https://formspree.io/f/myzgkjna`).
